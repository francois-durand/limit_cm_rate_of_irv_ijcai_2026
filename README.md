Presentation
------------
The code of this repo was used to prepare the article:

Élie de Panafieu, François Durand, and Guillem Perarnau. Super Condorcet Winners and the Limit Coalitionaly Manipulability of IRV. In IJCAI 2026.

Contents of the repo
--------------------

- __*.py__: Python files with the auxiliary functions.
- __simulations_curves.ipynb__: Python notebook, used to prepare the table and the plot of the article.
- __[simulations_curves.html](https://htmlpreview.github.io/?https://github.com/francois-durand/limit_cm_rate_of_irv_ijcai_2026/blob/main/simulations_curves.html)__: Notebook converted to html, especially useful for users with no Python distribution.
